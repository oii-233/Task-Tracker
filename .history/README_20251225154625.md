# Task Tracker Application

A simple **Task Tracker App** built with **React** and **Vite**.

## Features

- Add tasks
- Mark tasks as completed
- Delete tasks
- Dark mode (saved in localStorage)
- Task statistics page
- Responsive design

---

## Setup

```bash
git clone https://github.com/oii-233/Task-Tracker.git
cd Task-Tracker-Application
npm install
npm run dev
```
