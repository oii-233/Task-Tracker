# Task Tracker Application

A simple and responsive **Task Tracker** built with **React** and **Vite**.  
Features include task management, dark mode, and a statistics page.

---

## ðŸ›  Features

- **Add Tasks**  
  Add new tasks with a text input or by pressing **Enter**.

- **Mark Completed**  
  Toggle tasks as completed with a checkbox.

- **Delete Tasks**  
  Remove tasks instantly from the list.

- **Dark Mode**  
  Switch between light and dark themes; preference is saved in `localStorage`.

- **Stats Page**  
  View total tasks, completed tasks, and remaining tasks.

- **Persistent Data**  
  All tasks and dark mode preference are saved in `localStorage` and persist after refresh.

- **Responsive Design**  
  Works well on desktop and mobile screens.
